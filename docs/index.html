VulgaScience Publisher

VulgaScience Publisher is a creator tool for preparing original educational TikTok video drafts. It generates a script, voiceover, subtitles and a vertical video, then uploads the draft to TikTok inbox for final creator review before posting.

  • Creates educational short-form video drafts.
  • Checks format, audio and subtitle quality before upload.
  • Uses TikTok Content Posting API with creator-controlled final publishing.
